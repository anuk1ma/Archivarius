# Notes for Report and Presentation

## Идея проекта

`Архивариус` — это концептуальный веб-сайт для поиска и условного заказа книг мировой классики. Проект стилизован под старую библиотеку и ориентирован на сочетание визуальной атмосферы и основных full-stack функций.

## Что можно показать на защите

- главная страница и концепция дизайна
- каталог с поиском, фильтрами и сортировкой
- регистрацию и вход
- работу избранного и корзины
- оформление заказа
- профиль с историей заказов
- админ-панель
- структуру БД и REST API

## Сущности БД

- users
- books
- favorites
- cart_items
- orders
- order_items

## Почему проект подходит под критерии

- есть React и маршрутизация
- есть Node.js API
- есть PostgreSQL
- реализована авторизация и роли
- есть CRUD у администратора
- есть 404 и обработка ошибок
- предусмотрена документация и деплой
